一、PMP 简介

PMP指的是项目管理专业人士资格认证。

它是由美国项目管理协会（Project Management Institute(PMI)发起的，

严格评估项目管理人员知识技能是否具有高品质的资格认证考试。

其目的是为了给项目管理人员提供统一的行业标准。
